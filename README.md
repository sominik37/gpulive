## 🚀 Project Summary

CGPUlive is a premium, enterprise-grade GPU cloud provider designed to empower developers, AI researchers, and businesses with scalable, high-performance compute resources. Built with a focus on speed, reliability, and ease of use, our platform offers a seamless experience from resource discovery to deployment.

## ✨ Key Features

- 🖥️ **GPU Inventory**: Real-time tracking and availability of industry-leading GPUs (NVIDIA H100, A100, L40S, etc.).
- 📅 **Resource Reservation**: Advanced booking system for on-demand and long-term compute needs.
- 🌐 **Global Data Centers**: Tier-3 and Tier-4 infrastructure optimized for low latency and high throughput.
- 🧠 **AI Solutions**: Specialized clusters for model training, inference, and large-scale research projects.
- 💎 **Premium UI/UX**: Modern dashboard with glassmorphism, fluid animations (Motion), and bento-grid layouts.
- ⚡ **Lightning Fast**: Optimized with Vite and React 19 for instantaneous performance.

## 🛠️ Tech Stack

- **Framework**: [React 19](https://react.dev/) (Client-side)
- **Tooling**: [Vite 6](https://vitejs.dev/) for ultra-fast builds and HMR.
- **Styling**: [Tailwind CSS 4](https://tailwindcss.com/) with custom design tokens.
- **Animations**: [Motion](https://motion.dev/) (formerly Framer Motion) for smooth, interactive micro-animations.
- **Icons**: [Lucide React](https://lucide.dev/) for a consistent, modern icon set.
- **Routing**: [React Router DOM 7](https://reactrouter.com/) for multi-page navigation.
- **Language**: [TypeScript](https://www.typescriptlang.org/) for type-safe development.
